# AutoShare

Free Nepal MeroShare IPO auto-apply platform. Public, open-source.

Users connect their own GitHub Actions — the bot runs on their account, applies for IPOs automatically using their credentials. No server required on your end.

## How it works
1. Fork or use the public workflow
2. Add your MeroShare credentials as GitHub Secrets
3. Bot applies for every open IPO automatically

## Stack
- Python
- FastAPI (backend)
- GitHub Actions (user-side bot)
- GCP e2-micro
